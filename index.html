Aprendizajes C++

¿Que es?

La intención de su creación fue el extender al exitoso lenguaje de programación C
con mecanismos que permitieran la manipulación de objetos.Posteriormente se añadieron
facilidades de programación genérica, que se sumaron a los paradigmas de programación
estructurada y programación orientada a objetos.

C ++ es un lenguaje de programación orientado a objetos de alto nivel que ayuda a los
programadores a escribir programas rápidos y portátiles.


Usos de c++

Las aplicaciones del lenguaje C++ son muy extensas. Podemos nombrar que navega-
dores WEB, Sistemas operativos, Bases de datos, bibliotecas, aplicaciones gráficas, nubes,
videojuegos, compiladores, etc.

Por todos estos usos y aplicaciones podemos concluir que la importancia del lenguaje C++
es muy grande y está presente en muchos sitios.C++ nos permite, entre otras muchas cosas:

- Crear datos complejos.
- Definir operaciones sobre los datos complejos.
- Relacionar los datos complejos entre ellos.
- Implementar múltiples patrones de diseño.


Ventajas de c++

- C++ es un lenguaje que es didáctico, y es fácil de usar y de entender, para algunas
personas es más cómo usar este lenguaje de programación, gracias a sus fáciles conceptos y
los diversos métodos que tiene para usar este lenguaje.

- Es un lenguaje que constantemente se logra actualizar, para lograr una fácil programación,
qué permitaal usuario entender fácilmente lo que está programando y que tenga claro que
quiere programar, y para qué quiere programar.>


Desventajas de c++

- No es recomendable para desarrollo de páginas Web.

- Tiene un extenso lenguaje, lo cual puede confundir de cierto modo a la persona que está
usando el leguaje de programación de c++.

- En algunos casos, se deben usar demasiadas líneas de códigos, lo cual puede resultar
ser aburrido para la persona qu eestá usando esta aplicación con el fin de programar.
